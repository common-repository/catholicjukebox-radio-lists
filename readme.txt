Show the Recently Played or Top 5 Requested list from CatholicJukebox.com Radio

== Description ==

This widget gives you the ability to show the Recently Played or Top 5 Requests of the past 7 days 
for CatholicJukebox.com Radio, located at http://www.catholicjukebox.com - the listings include the 
cover art if available, along with a link to the artist's song listing page so listeners can make requests
online. There is a link at the bottom of the list to launch the radio station webpage. 

== Installation ==

1. Upload `cjbw.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. On the widget admin page, select if you want the Recently Played and/or the Top 5 Request lists.

== Frequently Asked Questions ==
= Do I need to have an account on CatholicJukebox.com?  =

No, the plugin does not require any login.

= How often do the lists change =

The lists are generated with each song that is played on the streaming radio. The top 5 list will always represent the past 7 days from that point.
